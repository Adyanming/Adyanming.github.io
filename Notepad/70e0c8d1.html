Local gulp not found in ~\Desktop\xxx

  • 这表示在 ~\Desktop\xxx 这个项目目录中，Gulp 工具没有被安装。

Try running: npm install gulp

  • 这是一个建议，提示你可以通过运行 npm install gulp 来安装 Gulp，这样本地的 Gulp 工具就会被正确安装到项目的 node_modules 目录中。
